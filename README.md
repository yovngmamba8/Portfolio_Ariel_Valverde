## 🎯 Sobre el Proyecto

Este portafolio no es solo una página estática, sino una aplicación web moderna construida con principios de **diseño atómico y componentización (DRY)**. El objetivo principal fue lograr un rendimiento ultrarrápido (gracias a la arquitectura de islas de Astro) y un diseño limpio e intuitivo, reflejando mi visión de que el buen código debe ser seguro, escalable y generar un impacto real.

### ✨ Características Principales
- **Arquitectura Modular:** Uso intensivo de componentes `.astro` (Cards, UI elements, Layouts) para un código altamente mantenible.
- **Rendimiento Óptimo:** Aprovechamiento de Astro para generar HTML estático y reducir la carga de JavaScript en el cliente.
- **Diseño Responsivo:** Interfaz adaptada a cualquier dispositivo (Mobile First) utilizando Tailwind CSS.
- **Integración CDN:** Uso de Devicon para la carga eficiente de iconografía de tecnologías.
